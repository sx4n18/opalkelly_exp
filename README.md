# Opal Kelly Experiment

This project will mainly log down all the experiment I have done with the purchased opal Kelly board.

Board model: XEM7310-A75

It is operating in a way where we do not have to dig deep into the very low level of USB 3 to make the board talk to the host PC.

But it has some learning curve and RTL needs to be packed in a specific way.

I will start by doing the most basics and drive and monitor the hardware with Python.

Maybe GUI can also be scripted using Python later on?
